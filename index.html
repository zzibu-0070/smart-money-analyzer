분석 알고리즘 전면 강화 + 자동 검증·교정 엔진
▸ 설계자 행동 해석 (신규)
규칙 I — Theta Burn & L자형 횡보 — 거래량 없는 횡보 지속 시 "설계자의 옵션 프리미엄 소멸 작전"으로 우선 해석. 판별 기준 3가지 + 리테일 지치게 만들기 패턴 탐지.
규칙 J — 옵션 핀닝 정밀도 — 종가가 주요 행사가와 ±0.1% 이내 = MM 델타 헤지 비용 최적화(핀닝 성공). 실패가 아닌 설계 의도로 해석.
규칙 K — Final Absorption(최종 흡수) — 하락 끝단 블록딜 + 가격 밀림 없음 조합 = 설계자 바닥 다지기 완료 신호. 5가지 판별 기준 포함.
규칙 L — 거시 유동성(Financial Plumbing) — 분석 시점에 web_search로 준비금·RRP·SOFR-IORB·DXY 자동 검색. 유동성 우호/제한 환경 판별 후 SECTION 5 시나리오 확률에 가중치 반영.
규칙 M — CTB 낮은 환경의 역설 — CTB Fee 낮음 + 가용잔고 감소 = 설계자의 저비용 가격 억제 선점 경고. 기술적 반등 신호만으로 진입 시 무한 핀 리스크 경고 의무화.
▸ 오류 수정 (데이터 계산)
규칙 A-2 — Total OBV 자동 교정 — 전체 OBV = 기관+프로+리테일 합산 공식 명시. 이미지 Total 탭과 불일치 시 계산값 우선 적용, 런타임에 자동 교정 후 배지 표시.
규칙 B-2 — 옵션 체인 전 스트라이크 누락 방지 — $38 콜 OI 1,606 등 OTM 대형 OI를 인근 스트라이크로 합산·분산 금지. 스트라이크 수 15개 미만 시 재검토 지시.
규칙 C-2 — 롤오버 잔여 OI 판별 — 딥 OTM(-30% 이상) + Volume=0 + OI 10배 이상 = 롤오버 잔여물 판별, Max Pain 계산 제외. GEX 차트엔 표시하되 주석 부착.
규칙 C-3 — Max Pain 카드 일관성 — 카드는 반드시 롤오버 제외 ATM 기준값 표시. 런타임에 현재가 대비 -30% 이상 이격 시 "롤오버 포함 가능성" 경고 자동 삽입.
규칙 C-4 — OI P/C 롤오버 제외 계산 — P/C 산출 시 롤오버 OI 제외 의무화. 2.0 초과 시 경고 배지 자동 표시.
규칙 D-2 — 가용잔고 변화율 기준 — 절댓값 아닌 전일 대비 Δ% 기준으로 해석. 950K→300K(-68%)를 "충분"으로 오독하는 오류 방지.
규칙 F — 볼륨 P/C 만기일 보정 — 만기일 당일 볼륨 P/C는 롤오버로 구조적 과장. 전일 추세와 비교 해석 필수, 단독 방향성 신호 금지.
▸ 기능 추가
규칙 G — 프로 OBV + Short% 조합 해석 — 프로 OBV 급반전 + Short% 급락(-5%p↑) 동시 발생 시 숏 커버링으로 우선 해석. "물량 이전" 추상 표현 오용 방지.
규칙 H — OTM IV 급등 → 이진 이벤트 탐지 — 현재가 +20% 이상 OTM 콜 IV가 ATM 대비 50%p↑ 이면 이벤트 프라이싱 신호 식별 및 이벤트 특정 의무화.
SECTION 0 — 뉴스·이벤트·공시 자동 검색 — web_search 3회 검색: ①최근 7일 뉴스 ②향후 1개월 이벤트(RSA·기술 컨퍼런스 포함, 과거 이벤트 제외) ③최근 SEC 공시. 3분할 카드로 표시.
Phase 날짜 거래일 보정 — NYSE 2026 휴장일·주말 목록 내장. Phase 날짜에 비거래일 포함 시 런타임 자동 경고 배지 삽입.
자동 검증·교정 인젝터 — API 응답 HTML에 검증 스크립트 주입. OBV 산술 교정·Max Pain 경고·P/C 경고·Phase 날짜 검증 4종 런타임 실행.
▸ UI 개선
Section 6 폰트 통일 — 3줄 요약 .summary-row 클래스 적용. ①②③ 마커 레드(#CF222B) 강조.
GEX 전체 스트라이크 범위 의무화 — 현재가 근방만 넣는 오류 방지. 스트라이크 수 기반 차트 높이 자동 조정.
섹션 헤더 핵심 포인트 배지 — 각 섹션 제목 옆 kp-pos/neg/warn/info 배지로 핵심 발견 표시.
업데이트 로그 버튼 — 헤더에 📋 버튼 추가. 모달로 전체 변경 이력 확인 가능.
치코리타 파비콘 — 배경 투명 처리 후 base64 임베드. 외부 파일 없이 단독 실행 가능.
